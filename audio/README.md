# Audio - In Mihais Küche (A1 vocabular)

## Flashcards (32 WAV)
Locație: `audio/letters/*.wav`
Voce: Microsoft Hedda Desktop (de-DE), Rate -1.
Generare: PowerShell System.Speech (`SpeechSynthesizer`).

Cele 32 fișiere corespund 1:1 cardurilor din `flashcards.js`:
- 8 Verpackungen (flasche-wasser, tafel-schokolade, sack-kartoffeln, dose-tomaten, glas-marmelade, tuete-chips, becher-joghurt, tube-senf)
- 6 Hauptgerichte (schnitzel, bratwurst, gulasch, braten, eintopf, kartoffelpueree)
- 4 Suppen (gemuesesuppe, tomatensuppe, huehnersuppe, linsensuppe)
- 5 Desserts (eis, kuchen, apfelstrudel, pudding, pfannkuchen)
- 6 Küchengeräte (topf, pfanne, messer, gabel, loeffel, backofen)
- 3 Verben (kochen-braten-backen, schneiden, schaelen-wuerzen)

## MP3-uri teorie (7 subsecțiuni — de înregistrat de Claudia)
- `01-intro.mp3` — Mihai ne primește în bucătărie
- `02-verpackungen.mp3` — Cele 12 ambalaje (cum ceri la magazin)
- `03-hauptgerichte.mp3` — Feluri principale
- `04-suppen.mp3` — Supe (die ...suppe)
- `05-nachtisch.mp3` — Deserturi
- `06-kuechengeraete.mp3` — Ustensile și vase
- `07-verben-sumar.mp3` — Verbe de gătit + sumar

## Sursă
Vocabular A1 standard de bucătărie. Tema Verpackungen pornită de la imaginile DeutschFuchs trimise de Claudia (eine Packung, ein Sack, eine Flasche, ein Netz, eine Dose, eine Tüte, eine Kiste, eine Tafel, ein Glas, eine Tube, ein Becher, eine Schale) + extensii proprii (feluri principale, supe, deserturi, ustensile, verbe).

## Notă
Fișă **de sine stătătoare** (reutilizabilă ca dicționar tematic). Cast: Mihai = Koch / protagonist (gătește în restaurantul lui), Andreea co-narator. Capcana cheie predată: ambalaj + produs FĂRĂ „von" (eine Flasche Wasser, NU eine Flasche von Wasser).
